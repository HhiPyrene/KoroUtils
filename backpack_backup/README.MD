# BackpackBackup 背包备份
### 配置文件说明
已下为默认值，缺省即使用默认值 

BackpackBackup.setting:
```properties
#最大单人可储存背包数
max-backup = 5
#指令开头名
header = backup
#备份指令名
backup = backup
#恢复指令名
restore = restore
#删除指令名
remove = remove
#打印列表指令名
list = list
#是否清除已使用的备份
delete-used-backup = true
```
CommandEN.setting:
```properties
#指令集启用
BackpackBackup = true
```
